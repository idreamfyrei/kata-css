Capabilities  /  01
01
Runtime Transform
Dynamic style injection at runtime. Scans kata-* classes on DOMContentLoaded and converts them to inline styles instantly.
02
Zero Config
Drop it in and start coding. No build steps, no config files, no PostCSS, no purge setup. One script tag.
03
CDN Ready
Serve via jsDelivr or npm. Works in static HTML, React, Vue, Svelte — anywhere you write a class attribute.
04
Chai Color Scheme
50+ color tokens named after real chai and tea varieties. A ChaiCode cohort original. Works with text, bg, and border.
